# 🎵 Moodix - Mood-Based Music Player  

Moodix is an intelligent music player that **curates playlists based on your mood!** Using AI-driven mood detection, it analyzes **facial expressions, text input, or voice tone** to suggest the perfect tracks for your emotions.
